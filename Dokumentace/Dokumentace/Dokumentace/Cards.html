Cards Module Documentation

Overview

This module defines several classes and initializes multiple lists of game assets for a role-playing or board game, including characters, enemies, objects, magic objects, places, and spells.

Characters

Characters are pivotal game entities with attributes defining their capabilities and starting conditions.

  • name (str): The character's name.
  • image (str): Path to the character's image file.
  • strength (int): Measures the character's physical power.
  • craft (int): Measures the character's intelligence or magical ability.
  • life (int): Indicates the number of life points a character has.
  • fate (int): Represents the fate points that may be used for rerolling dice.
  • gold (int): The initial amount of gold the character possesses.
  • alignment (str): Describes the moral and ethical perspective of the character ('Good', 'Neutral', 'Evil').
  • start (str): The starting location of the character on the game board.

Enemies

Enemies challenge the players throughout the game, appearing in battles and affecting player progress.

  • name (str): Name of the enemy.
  • image (str): Path to the enemy's image.
  • effect (str): Description of the effect, currently a placeholder.
  • craft (int): The enemy's craft value, important for magical battles.
  • strength (int): The enemy's physical strength, used in combat.

Objects

Objects are items that players can collect and use within the game to improve their chances of success.

  • name (str): The name of the object.
  • image (str): Path to the object's image.
  • effect (str): Describes how the object affects the game or the player.
  • effect_value (int, optional): Quantifies the effect's impact.

Magic Objects

Magic objects provide significant advantages and can alter the game's dynamics dramatically.

  • name (str): The name of the magic object.
  • image (str): Path to the object's image.
  • effect (str): Type of magic effect.
  • effect_value (int, optional): The magnitude of the magical effect.

Places

Places are specific locations on the game board where players can interact and experience various game mechanics.

  • name (str): The name of the place.
  • image (str): Path to the place's image.
  • effect (str): Describes what happens at this location.

Spells

Spells allow characters to perform magical actions that can affect themselves, other players, or the game state.

  • name (str): The name of the spell.
  • image (str): Path to the spell's image.
  • effect (str): The magical effect of the spell.
